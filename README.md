# OC版本 > 0.7.2

## 那些功能正常

1. CPU睿频
2. 核显解码（目前显示正常，但是操作F1功能会闪一下）
3. 风扇控制（isata menus）
4. 板载网卡 + wifi + 蓝牙
5. 休眠
6. USB
7. 万能声卡
